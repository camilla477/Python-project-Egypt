📊 Egypt Protest Analysis

Welcome

This project analyzes protest events in Egypt using ACLED (Armed Conflict Location and Event Data) to understand trends, patterns, and impacts of demonstrations.

View the Analysis

The main analysis is in bella_copia.ipynb. Choose a viewing option:

  • View on nbviewer 📓 (Recommended - best notebook rendering)
  • View on GitHub 🐙 (Basic notebook preview)
  • View locally: 
    git clone https://github.com/camilla477/draft_egypt_protests.git
cd draft_egypt_protests
jupyter notebook bella_copia.ipynb

Datasets

  • ACLED_egypt2011.csv - 2011 Protest Events
  • ACLED_alsisi.csv - Alsisi Period Protest Events

Project Files

  • bella_copia.ipynb - Main analysis notebook
  • ACLED data files (.csv)
  • Visualization charts (.png)
